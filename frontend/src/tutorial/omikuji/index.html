目標

ボタンを押すとおみくじの結果が出てくるプログラミングを作成しよう。

次のページからはヒントなので自信のある人はヒントを見ないでやってみよう！

例

ボタン

 まずはボタンを作ってみましょう。ボタンを作るにはHTML Elementsブロックの中から、<button>で囲まれたブロックを選びましょう。
 ボタンの中に文字を入れてみましょう、ボタンを作ることができたら結果を表示する場所を作りましょう。結果の中身を後で変えることができるようにidを つけて要素を取得できるようにしましょう。中身は後でjavascriptを用いて変更するので中身はからでも構いません

Javascript

 javascriptのコードをscriptブロックの中にJavaScriptブロックをいれることで 作りましょう。どんな結果になるのかはおみくじなので運で決まってほしいです。そこで ここではMathブロックに含まれている、1未満の正の実数をランダムで生成するブロックを 使ってみましょう。この場合0~0.20の数字が生成される確率は20%です。発生した乱数はそのままでは使えないので 変数に保存しましょう。変数はVariablesブロックの変数の作成をクリックすると変数を作ることができます。 次に条件分岐を作りましょう。条件分岐を作るにはLogicブロックの「もし」と書かれているブロックを使ってみましょう。条件分岐と 条件を満たすかどうかによって実行するプログラムを変化させることです。

結果の文字やボタンの色、文字の大きさ表示される位置などを変更してみましょう。 cssブロックを使うことでこれらの変更を加えることができます。
