構造

Todoリストを構成する基本的な要素として、

  • Todoリストが入るタグを作る
  • Todoリストにテキストを入れられるようにする
  • Todoリストを追加、削除するボタンを作る
この３つが挙げられます。
まずリストを作る元となる<ul>タグと、テキストを入れる場所を作る<input>タグを<body>タグに入れてみましょう。

IDでタグ付け

次に、<ul>と<input>にIDブロックをつけましょう。
IDをつけることによって、この後行うJavaScriptを用いた作業でこれらのタグに色々な機能をつけることができます。
IDの名前は今回は<ul>にtodo-list、<input>にtodo-inputと入れることにしましょう。

次に、押すことでTodoリストを追加できるようなボタンを作りましょう。
ボタンを押した後に

  • <ul>の中に新たな項目<li>を入れる
  • <input>の中に入れた文章を<li>の中に入れる
この２つがしたいですね。
ここでJavascriptを用いてクリックしたら
  • 要素名liを追加する
  • 追加した要素の内容を<input>の内容に変更する
ようにしましょう。

次に、削除ボタンをTodoリストに追加しましょう。
ボタンを押した後に、

  • 要素buttonを追加し、その内容を「削除」に変える
  • 削除ボタンが押された際に<li>タグを消す
とすればいいですね。

最後に、作ったTodoリストを編集できるようにしましょう。
削除ボタンを作ったときと同様に編集ボタンを作ります。
