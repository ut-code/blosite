CSS

 今まではhtmlとは何かについて学びました。htmlがウェブサイトの構造を表す言語だったのに対してCSSはウェブサイトの見た目を
表す言語です。blositeではstyle属性をhtml要素に加えることでCSSを記述して見た目を変更することができます。style属性を追加する
には左上のHTMLAttributesの中に入っているstyleと書かれたブロックを使います。

styleブロックの使い方

 styleブロックの中には歯車が入っています。その歯車をクリックすると右の画像のように項目がでてきます。項目と書かれたブロックをドラッグすることで接続する部分の数を変更することができます。

色を変えてみよう

 まずは実際に文字の色を変えてみましょう。HTMLを知ろうのところで作成した遠足の持ち物リス
トがあるのでその文字の色を変えてみましょう。 文字の色を変えるにはstyleブロックにCSSカテゴリーに含まれているcolorブロックを使います。色は色コードを入力するか色の名前を英語で入力することで変更することができます。
